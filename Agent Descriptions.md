# Agent Specifications

## 1. Supervisor Agent
- Directs workflow  
- Merges agent outputs  
- Maintains global memory  

## 2. Task Intake Agent
- Classifies tasks  
- Extracts deadlines  
- Assigns complexity levels  

## 3. Task Decomposition Agent
- Breaks long tasks into subtasks  
- Assigns duration & effort  

## 4. Scheduling Agent
- Balances workload across 7 days  
- Avoids overload  
- Deadline-aware planning  

## 5. Analytics Agent
- Computes weekly load  
- Suggests improvements  
- Generates user insights
